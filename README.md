# Real-World Data Project: Sales Prediction using Advertising Data

## Project Overview
This project is a real-world data analysis and machine learning project based on advertising data. It analyzes the relationship between advertisement spending and sales performance.

## Domain
Retail / Marketing Analytics

## Objective
- Analyze advertising budget impact on sales
- Perform Exploratory Data Analysis (EDA)
- Build a predictive model using Machine Learning
- Visualize patterns and insights

## Dataset Features
- TV Ad Budget
- Radio Ad Budget
- Newspaper Ad Budget
- Sales

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow

### 1. Data Loading
Dataset is loaded using Pandas.

### 2. Data Cleaning
- Checked missing values
- Removed duplicates
- Removed unnecessary columns

### 3. Exploratory Data Analysis (EDA)
- Histograms for distribution
- Heatmap for correlation
- Scatter plots for relationships

### 4. Feature Selection
- X = Advertising budgets
- y = Sales

### 5. Model Building
- Linear Regression model used
- Train-test split applied

### 6. Prediction
- Sales predicted using test data

### 7. Evaluation
- Model performance evaluated using error metrics

## Key Insights
- TV advertising has strong impact on sales
- Radio advertising shows moderate impact
- Newspaper advertising has less impact
- Clear correlation exists between ad spend and sales

## Conclusion
This project demonstrates how machine learning can be used in real-world marketing to predict sales and improve business decisions.
